# 💊 DrugPharmers - Top 300 Drugs Study Platform

**DrugPharmers** is a robust, full-stack web application meticulously designed to help pharmacy students master the Top 300 drugs. Merging clinical education with modern technology, the platform features highly customizable digital flashcards and an interactive study mini-game to turn complex pharmacology data into an intuitive, high-retention learning experience.

## 🎯 Project Overview

Pharmacy school demands rapid, accurate memorization of vast amounts of drug information. **DrugPharmers** provides an interactive alternative to static paper cards, allowing students to filter study sets dynamically and gamify their learning to lock down critical data points before exams or rotations.

### Key Features

- **Customizable Flashcards**: Study the top 300 drugs with customizable filtering (filter by Drug Class, Brand vs. Generic, Core Indications, or High-Alert status).
- **Interactive Mini-Game**: A high-speed, timed review game designed to build pattern recognition for rapid drug matching (e.g., matching mechanisms of action to brand names).
- **NextAuth.js Authentication**: Secure user login pipelines utilizing individual developer profiles or custom credentials backed securely by Django.
- **Comprehensive Drug Profile Database**: Detailed tracking for every compound, including:
  - Brand & Generic naming conventions
  - Pharmacological classifications & mechanisms of action (MOA)
  - Core clinical indications & FDA-approved uses
  - Major adverse drug reactions (ADRs) & critical black box warnings
